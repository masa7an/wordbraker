# Word Breaker（英単語ブロック崩し）

英単語学習とブロック崩しを組み合わせた教育ゲームです。

## ゲームの目的

英単語の意味を3択から選び、正解ブロックを破壊してステージをクリアしましょう！

## ゲームルール

1. **目的**: 英単語の意味を3択から選び、正解ブロックを破壊してステージをクリア
2. **正解ブロック**: 2ヒットで破壊可能（HP: 2、1ヒット目で色が変わる）
3. **不正解ブロック**: 完全無敵（壊れない）
4. **扉**: 全正解ブロック破壊でアンロック、ボールが当たるとクリア
5. **ライフ**: 初期10、ボール落下で-1
6. **スコア**: 正解ブロック破壊で+100、コンボ・ノーミスボーナスあり

## 操作方法

### マウス操作
- **パドル移動**: マウスで移動
- **ボール発射**: クリック（ステージ開始時、リスポーン後）

### ゲームパッド操作
- **パドル移動**: 十字キー（左右）または左アナログスティック
- **ボール発射**: Aボタン（ステージ開始時、リスポーン後、各種アクション）

### キーボード操作
- **Hキー**: ハードモード切り替え（いつでも）
- **Escキー**: 終了（リザルト画面で）

## ハードモード

- **Hキー**でいつでも切り替え可能
- 正解ブロックも赤色で表示されるため、見分けがつかない
- リザルト画面でHキーを押すとハードモードで再挑戦

## スコアシステム

- **基本スコア**: 正解ブロック破壊で+100
- **コンボ**: 3回連続で破壊すると×1.2倍
- **ノーミスボーナス**: ミスなしでクリアすると×1.2倍 + "PERFECT!!"表示
