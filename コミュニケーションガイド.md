# コミュニケーションガイド

## 現在の形式（問題なし）

現在の形式（①調整、②質問など）は**十分に分かりやすく、推奨される方法**です。

## より効率的なコミュニケーション方法の提案

### 1. **優先順位の明示**（推奨度：★★★★★）

```
【緊急】ブロックの幅を3倍に
【要確認】スコア計算のロジック
【改善提案】UIの色を調整
```

### 2. **カテゴリ分け**（推奨度：★★★★☆）

```
【バグ修正】
- ブロックの幅を3倍に
- ボールが画面外に出る問題

【機能追加】
- 音声再生機能
- ランキング表示

【調整】
- スコア表示位置
- フォントサイズ
```

### 3. **具体的な数値指定**（推奨度：★★★★★）

```
【調整】
- ブロック幅: 100 → 300
- ボール速度: 4 → 6
- フォントサイズ: 24 → 32
```

### 4. **期待する動作の説明**（推奨度：★★★★☆）

```
【要修正】
- 現在: 文字がはみ出ている
- 期待: ブロック内に文字が収まる
- 方法: ブロック幅を3倍に
```

### 5. **複数項目の整理**（推奨度：★★★★★）

```
【調整】
1. ブロック幅を3倍に（文字はみ出し対応）
2. スコア表示位置を右上に移動
3. フォントサイズを32pxに統一

【質問】
1. この形式で問題ないか？
2. 他に改善点はあるか？
```
